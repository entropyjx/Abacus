```
stack.pop(); // Remove from head
stack.peek(); // Return ref to head not remove
```

Queue
```
Deque<Integer> queue = new ArrayDequeue<>();
queue.add(12); // Add to tail
queue.remove(); // Remove from head
queue.peek(); // Return ref to head, not remove
```
